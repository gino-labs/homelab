# Homelab as Code

## Services

- Gateway router (Firewalld, NetworkManager)
- Dnsmasq (DNS, DHCP, TFTP/PXE)
- Nginx reverse proxy
- FreeIPA server
- Repo server
- Podman host (GitLab, Splunk, etc)
- NFS server
- SMB server
- Qemu/KVM Hypervisor (2 VMs)

### Other configurations

- SSH hardening
- DNF automatic
- Fail2ban
- ClamAV
- Nessus Essentials
